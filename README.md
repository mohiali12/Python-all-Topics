Python Concepts Repository
This repository contains a collection of Jupyter Notebooks and datasets designed to help understand fundamental to intermediate Python concepts. It is structured for beginners and learners who want hands-on practice with core programming and data handling topics.


Repository Structure
Core Python Concepts
fundamentals.ipynb → Basic syntax, variables, data types
function.ipynb → Functions, parameters, return values
ForLoop.ipynb → Looping concepts and iterations
Classes and Objects.ipynb → Object-Oriented Programming (OOP)


Data Handling & Libraries
Numpy.ipynb → Numerical operations using NumPy
Pandas.ipynb → Data analysis using Pandas
Pandas-Mohi.ipynb → Additional Pandas practice
Array.ipynb → Array operations and usage
Slicing.ipynb → Data slicing techniques


Visualization
matplot.ipynb → Data visualization using Matplotlib
Data Files
New_Raw_Data.csv → Sample dataset for practice
my_data.csv → Custom dataset
sharma-kohli.csv → Example dataset for analysis


Scripts & Web
python.py → Python script file
web.html → Basic HTML file (for integration/demo purposes)
